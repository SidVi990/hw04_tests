Вот что помогло мне написать эту страницу

  • Python - это мощный язык программирования, на котором можно создавать самые разные приложения, не зависящие от платформы.
  • Django — является чрезвычайно популярным и полнофункциональным серверным веб-фреймворком, написанным на Python. Проект поддерживается организацией Django Software Foundation.
  • HTML - (HyperText Markup Language — «язык гипертекстовой разметки») — самый базовый строительный блок Веба. Он определяет содержание и структуру веб-контента.
  • Bootstrap — Самая популярная библиотека HTML, CSS и JS в мире.

Проект создан в рамках обучения backend-разработке на платформе Яндекс практикум.
